# markdown-setting

[Markdown Preview Enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced) の拡張機能の設定ファイル

```
::: info
info
:::

::: success
success
:::

::: warning
warning
:::

::: danger
danger
:::
```

上記の記法に対応したパーサーとスタイルを作成した

## 設定方法

`~/.crossnote`配下の `style.less` と `parser.js`を上書きする。
